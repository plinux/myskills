# Glossary: Operations, Passenger Service And Safety

用于运营组织、客运服务、票务、行车计划、试运行、安全、应急和环境影响相关翻译。

| 中文术语 | 推荐英文 | 可选译法 | 适用语境/说明 |
| --- | --- | --- | --- |
| 运营 | operation | operations | 单概念用 operation，业务体系用 operations。 |
| 运营管理 | operations management | operation management | 管理体系。 |
| 运营组织 | train operation organization | operations planning | 行车组织。 |
| 列车运行图 | train diagram | working timetable | 图形计划用 train diagram，时刻计划用 working timetable。 |
| 列车时刻表 | timetable | train timetable | 乘客或运营计划。 |
| 行车计划 | train service plan | operating plan | 运营组织。 |
| 运行时间 | running time | travel time | 区间运行或全程旅行需区分。 |
| 停站时间 | dwell time | station dwell time | 站台停留。 |
| 折返时间 | turnback time | turnaround time | 终端作业。 |
| 发车间隔 | headway | service interval | 客运运营。 |
| 最小行车间隔 | minimum headway | minimum train interval | 能力计算。 |
| 运输能力 | transport capacity | carrying capacity | 系统能力。 |
| 通过能力 | line capacity | throughput capacity | 线路/区间能力。 |
| 高峰小时 | peak hour | peak period | 客流和运能。 |
| 平峰 | off-peak period | off peak | 客运时段。 |
| 客流 | passenger flow | passenger volume | 流量或方向性。 |
| 客运量 | passenger volume | ridership | 统计口径。 |
| 断面客流 | section passenger flow | line-section passenger volume | 区间断面。 |
| OD 客流 | origin-destination passenger flow | OD flow | 起讫点客流。 |
| 客流预测 | passenger demand forecast | ridership forecast | 规划。 |
| 满载率 | load factor | crowding level | 车辆/列车载客。 |
| 列车满载率 | train load factor | train occupancy rate | 运营指标。 |
| 客流控制 | passenger flow control | crowd control | 车站组织。 |
| 限流 | passenger flow restriction | entry restriction | 客运组织。 |
| 乘降 | boarding and alighting | boarding/alighting | 站台过程。 |
| 换乘 | transfer | interchange | 乘客换线。 |
| 换乘站 | transfer station | interchange station | 城市轨道。 |
| 终点站 | terminal station | terminus | 线路终点。 |
| 中间站 | intermediate station | wayside station | 线路中间车站。 |
| 地下站 | underground station | subway station | 车站形式。 |
| 高架站 | elevated station | -- | 车站形式。 |
| 地面站 | at-grade station | ground-level station | 车站形式。 |
| 站台 | platform | station platform | 乘降区域。 |
| 岛式站台 | island platform | -- | 站台形式。 |
| 侧式站台 | side platform | -- | 站台形式。 |
| 站厅 | concourse | station hall | 车站公共区。 |
| 出入口 | entrance and exit | access point | 车站出入口。 |
| 通道 | passageway | corridor | 车站通行空间。 |
| 自动扶梯 | escalator | -- | 车站设备。 |
| 电梯 | lift | elevator | 英式 lift，美式 elevator。 |
| 无障碍设施 | barrier-free facilities | accessible facilities | 服务设施。 |
| 站台门 | platform screen door | PSD | 全高或系统总称。 |
| 安全门 | platform edge door | PED | 半高站台门常用。 |
| 自动售检票 | automatic fare collection | AFC | 系统。 |
| 自动售票机 | ticket vending machine | TVM | 票务设备。 |
| 闸机 | fare gate | ticket gate | 票务设备。 |
| 票卡 | fare medium | ticket card | 票务介质。 |
| 试运行 | trial operation | test operation | 开通前运营演练。 |
| 试运营 | initial operation | trial commercial operation | 中国项目常用需按法规核对。 |
| 联调联试 | integrated testing and commissioning | system integration test | 开通前测试。 |
| 调试 | commissioning | testing and commissioning | 工程交付。 |
| 验收 | acceptance | acceptance inspection | 工程/系统验收。 |
| 运营控制中心 | operation control centre | OCC; operations control center | 英式 centre，美式 center。 |
| 行车调度员 | train dispatcher | traffic controller | 运营控制岗位。 |

| 未遂事件 | near miss | close call | 安全管理。 |

| 排烟 | smoke extraction | smoke exhaust | 防灾通风。 |
| 防洪 | flood prevention | flood control | 运营防灾。 |
| 噪声 | noise | -- | 环境影响。 |
