# Glossary: Core Infrastructure

用于铁路工程、城市轨道交通工程、线路、轨道、路基、桥隧、车站和土建相关翻译。

| 中文术语 | 推荐英文 | 可选译法 | 适用语境/说明 |
| --- | --- | --- | --- |
| 铁路；铁道 | railway | railroad | 标准和国际技术文本优先 railway；北美可用 railroad。 |
| 铁路工程 | railway engineering | railway works | 工程建设、勘察设计和施工。 |
| 轨道交通 | rail transit | rail transport | 公共交通和行业泛称。 |
| 城市轨道交通 | urban rail transit | urban rail transport | GB/T 32852.3 标准名采用 Urban rail transit。 |
| 地铁 | metro | subway; underground railway | 国际技术文本优先 metro。 |
| 轻轨交通 | light rail transit | LRT; light rail | 城市轨道制式。 |
| 有轨电车 | tramway | tram; streetcar | 英式/国际常用 tram，北美乘客语境常用 streetcar。 |
| 单轨交通 | monorail transit | monorail | 城市轨道或专用制式。 |
| 市域铁路 | suburban railway | regional railway; commuter rail | 按项目定位选择。 |
| 城际铁路 | intercity railway | intercity rail | 城际客运。 |
| 高速铁路 | high-speed railway | high-speed rail | 技术和工程文本常用 high-speed railway。 |
| 重载铁路 | heavy haul railway | heavy-haul railway | 货运重载。 |
| 电气化铁路 | electrified railway | electric railway | 供电方式语境优先 electrified railway。 |
| 既有铁路 | existing railway | existing line | 改扩建和保护区语境。 |
| 新建铁路 | new railway | newly built railway | 工程建设。 |
| 改建铁路 | reconstructed railway | upgraded railway | 改扩建。 |
| 运营铁路 | operating railway | railway in operation | 运营状态。 |
| 干线 | main line | trunk line | 主干铁路。 |
| 支线 | branch line | branch railway | 支线铁路。 |
| 专用铁路 | private siding railway | dedicated railway | 企业或专用服务语境需核对。 |
| 铁路专用线 | railway siding | industrial siding; dedicated railway line | 接轨企业、货场等专用线。 |
| 货运专线 | freight dedicated line | freight-only line | 货运专用。 |
| 客运专线 | passenger dedicated line | passenger-only line | 客运专用。 |
| 客货共线铁路 | mixed passenger and freight railway | mixed-traffic railway | 客货混跑。 |
| 运营里程 | operating length | revenue length; operating distance | 客运/线网统计可用 revenue length。 |
| 建筑长度 | construction length | built length | 工程建设长度。 |
| 线路 | line | route; alignment; track | 见翻译规则。 |
| 正线 | main track | main line track | 车站/区间轨道。 |
| 到发线 | receiving-departure track | arrival-departure track | 车站股道。 |
| 折返线 | turnback track | reversing track | 城轨折返作业。 |
| 渡线 | crossover | scissors crossover | 两股道连通设备。 |
| 联络线 | connecting line | connection line | 线路间连接。 |
| 走行线 | running line | running track | 车辆段/场内或正线运行。 |
| 存车线 | stabling track | storage track | 停放列车。 |
| 车辆段 | depot | rolling stock depot | 城轨车辆基地。 |
| 停车场 | stabling yard | parking yard | 城轨停车场。 |
| 车站 | station | railway station | 客货运或城轨车站。 |
| 区间 | interstation section | section | 两站之间。 |
| 闭塞分区 | block section | block | 信号闭塞。 |
| 站场 | station yard | yard | 站内线路和设备范围。 |
| 编组站 | marshalling yard | classification yard | 货车编组。 |
| 驼峰 | hump | hump yard | 调车设备。 |
| 轨距 | track gauge | rail gauge | 两股钢轨工作边距离。 |
| 标准轨距 | standard gauge | -- | 1435 mm 语境。 |
| 宽轨 | broad gauge | -- | 大于标准轨距。 |
| 窄轨 | narrow gauge | -- | 小于标准轨距。 |
| 限界 | gauge | clearance | 必须按类型细化。 |
| 建筑限界 | structure gauge | construction clearance | 结构侵限控制。 |
| 车辆限界 | vehicle gauge | rolling stock gauge | 车辆轮廓。 |
| 装载限界 | loading gauge | loading clearance | 货物装载轮廓。 |
| 动态包络线 | kinematic envelope | dynamic envelope | 车辆运动状态包络。 |
| 轨道 | track | track structure | 工程结构，不等同于 rail。 |
| 钢轨 | rail | running rail | 钢轨本体。 |
| 护轨 | guard rail | check rail | 道岔/桥梁护轨。 |
| 道岔 | turnout | switch and crossing | 英式工程常见 switch and crossing。 |
| 尖轨 | switch rail | point blade | 道岔部件。 |
| 辙叉 | crossing | frog | 美式常见 frog。 |
| 岔枕 | switch sleeper | turnout sleeper | 道岔轨枕。 |
| 轨枕 | sleeper | tie | 英式 sleeper，美式 tie。 |
| 扣件 | fastening | rail fastening | 轨道扣件。 |
| 轨道板 | track slab | slab panel | 无砟轨道。 |
| 道床 | track bed | ballast bed | 有砟或无砟语境需区分。 |
| 道砟 | ballast | track ballast | 有砟轨道。 |
| 有砟轨道 | ballasted track | -- | 传统轨道结构。 |
| 无砟轨道 | ballastless track | slab track | 高铁/城轨常见。 |
| 无缝线路 | continuous welded rail | CWR | 长钢轨线路。 |
| 长钢轨 | long welded rail | LWR | 钢轨状态。 |
| 轨缝 | rail joint gap | joint gap | 钢轨接头。 |
| 轨道几何状态 | track geometry | track geometry condition | 检测和维护。 |
| 轨向 | alignment | lateral alignment | 轨道几何。 |
| 高低 | longitudinal level | profile | 轨道几何。 |
| 水平 | cross level | cant difference | 轨道几何。 |
| 轨距变化率 | gauge variation rate | gauge gradient | 轨道几何。 |
| 超高 | cant | superelevation | 曲线外轨抬高。 |
| 缓和曲线 | transition curve | spiral curve | 线路平面。 |
| 竖曲线 | vertical curve | -- | 线路纵断面。 |
| 路基 | subgrade | formation | 工程结构。 |
| 基床 | track formation | subgrade bed | 路基上部结构。 |
| 路堤 | embankment | fill | 填方。 |
| 路堑 | cutting | cut | 挖方。 |
| 边坡 | slope | side slope | 路基/隧道边坡。 |
| 桥梁 | bridge | railway bridge | 桥梁工程。 |
| 高架桥 | viaduct | elevated bridge | 高架区间。 |
| 涵洞 | culvert | -- | 小型过水结构。 |
| 隧道 | tunnel | railway tunnel | 隧道工程。 |
| 明挖法 | cut-and-cover method | open-cut method | 城轨/隧道施工。 |
| 盾构法 | shield tunnelling method | shield method | 盾构施工。 |
| 掘进机 | tunnel boring machine | TBM | 隧道机械。 |
| 衬砌 | lining | tunnel lining | 隧道结构。 |
| 洞门 | portal | tunnel portal | 隧道出入口。 |
| 联络通道 | cross passage | connection passage | 隧道安全疏散。 |
