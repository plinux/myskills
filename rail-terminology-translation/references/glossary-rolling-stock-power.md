# Glossary: Rolling Stock And Power

用于车辆、制动、轮轨关系、牵引传动、牵引供电、接触网和供电设施相关翻译。

| 中文术语 | 推荐英文 | 可选译法 | 适用语境/说明 |
| --- | --- | --- | --- |
| 机车车辆 | rolling stock | railway vehicles | 车辆总称。 |
| 车辆 | vehicle | car; rolling stock | 单车或车辆泛称。 |
| 列车 | train | trainset | 编组列车。 |
| 动车组 | electric multiple unit | EMU | 若为内燃动车组用 DMU。 |
| 内燃动车组 | diesel multiple unit | DMU | 动力形式。 |
| 机车 | locomotive | engine | 技术文本用 locomotive。 |
| 动车 | motor car | powered car | 编组内带动力车辆。 |
| 拖车 | trailer car | non-powered car | 无动力车辆。 |
| 头车 | cab car | driving car | 带司机室车辆。 |
| 车体 | car body | vehicle body | 车辆结构。 |
| 转向架 | bogie | truck | 国际/英式用 bogie，北美用 truck。 |
| 轮对 | wheelset | wheel set | 车轮和车轴总成。 |
| 车轮 | wheel | railway wheel | 车辆部件。 |
| 车轴 | axle | -- | 车辆部件。 |
| 轴箱 | axlebox | axle box | 轴承箱。 |
| 轮缘 | flange | wheel flange | 车轮轮缘。 |
| 踏面 | tread | wheel tread | 车轮踏面。 |
| 轮重 | wheel load | -- | 单轮载荷。 |
| 轴重 | axle load | axle weight | 轴载。 |
| 最大轴重 | maximum axle load | maximum allowable axle load | 限制参数。 |
| 车钩 | coupler | coupling | 车辆连接。 |
| 贯通道 | gangway | inter-car gangway | 车厢连接通道。 |
| 司机室 | driver's cab | cab | 驾驶室。 |
| 客室 | passenger compartment | saloon | 客运车辆内部空间。 |
| 一系悬挂 | primary suspension | -- | 转向架悬挂。 |
| 二系悬挂 | secondary suspension | -- | 转向架悬挂。 |
| 空气弹簧 | air spring | pneumatic spring | 悬挂部件。 |
| 减振器 | damper | shock absorber | 悬挂部件。 |
| 牵引系统 | traction system | traction drive system | 车辆牵引。 |
| 牵引电机 | traction motor | -- | 动力设备。 |
| 牵引变流器 | traction converter | traction inverter | 交流传动语境常用 traction converter。 |
| 辅助变流器 | auxiliary converter | auxiliary inverter | 辅助供电。 |
| 牵引逆变器 | traction inverter | -- | 逆变设备。 |
| 制动系统 | braking system | brake system | 车辆制动。 |
| 空气制动 | air brake | pneumatic brake | 压缩空气制动。 |
| 电空制动 | electro-pneumatic brake | EP brake | 制动控制。 |
| 再生制动 | regenerative braking | -- | 能量回馈。 |
| 电阻制动 | rheostatic braking | dynamic braking | 动力制动方式。 |
| 摩擦制动 | friction braking | mechanical braking | 闸瓦/盘式制动。 |
| 停放制动 | parking brake | holding brake | 停放状态。 |
| 紧急制动 | emergency braking | emergency brake | 安全制动。 |
| 防滑保护 | wheel slide protection | WSP | 制动防滑。 |
| 空转 | wheel slip | slip | 牵引状态。 |
| 滑行 | wheel slide | slide | 制动状态。 |
| 撒砂 | sanding | sand application | 提高黏着。 |
| 轮轨关系 | wheel-rail interaction | wheel-rail relationship | 接触力学。 |
| 轮轨接触 | wheel-rail contact | -- | 接触几何/力学。 |
| 黏着 | adhesion | wheel-rail adhesion | 牵引制动能力。 |
| 黏着系数 | adhesion coefficient | coefficient of adhesion | 轮轨黏着。 |
| 蛇行运动 | hunting motion | hunting | 车辆动力学。 |
| 限制速度 | speed restriction | restricted speed | 运营或线路限制。 |
| 牵引供电 | traction power supply | traction power supply system | 供电系统。 |
| 牵引变电所 | traction substation | traction power substation | 供电设施。 |
| 整流变电所 | rectifier substation | -- | 直流牵引供电。 |
| 分区所 | sectioning post | section post | 供电分区。 |
| 开闭所 | switching station | switching post | 供电开关设施。 |
| 接触网 | overhead contact line system | OCL; overhead catenary system; OCS | 英式/国际常用 OCL；北美常用 OCS。 |
| 架空接触线 | overhead contact line | contact wire | 系统或导线需区分。 |
| 接触线 | contact wire | trolley wire | 受电接触导线。 |
| 承力索 | catenary wire | messenger wire | 接触网承力。 |
| 吊弦 | dropper | hanger | 接触网部件。 |
| 定位装置 | steady arm | registration assembly | 接触网定位。 |
| 分段绝缘器 | section insulator | sectioning insulator | 接触网分段。 |
| 中性段 | neutral section | phase break | 交流供电。 |
| 受电弓 | pantograph | -- | 受流设备。 |
| 弓网关系 | pantograph-catenary interaction | pantograph-OCL interaction | 受流质量。 |
| 第三轨 | third rail | conductor rail | 城轨供电。 |
| 接触轨 | conductor rail | contact rail | 第三轨/接触轨。 |
| 回流 | return current | traction return current | 牵引供电。 |
| 迷流 | stray current | leakage current | 直流供电腐蚀风险。 |
| 接地 | earthing | grounding | 英式 earthing，美式 grounding。 |
| 钢轨电位 | rail potential | rail voltage | 牵引回流。 |
| 供电臂 | feeding section | feeder section | 牵引供电范围。 |
| 电分相 | phase separation | neutral section | 交流供电。 |
| 车辆限界 | vehicle gauge | rolling stock gauge | 与建筑/装载限界区分。 |
| 车载设备 | on-board equipment | onboard equipment | 信号/通信/车辆设备。 |
