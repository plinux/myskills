# Glossary: Signalling And Communication

用于铁路信号、城市轨道列控、通信、联锁、检测、调度和自动化系统相关翻译。

| 中文术语 | 推荐英文 | 可选译法 | 适用语境/说明 |
| --- | --- | --- | --- |
| 信号 | signalling | signal | 系统/专业用 signalling，单个信号机用 signal。 |
| 信号机 | signal | lineside signal | 轨旁固定信号。 |
| 信号显示 | signal aspect | aspect | 信号显示状态。 |
| 信号表示 | signal indication | indication | 向司机表达的信息。 |
| 进站信号机 | home signal | entrance signal | 进站控制。 |
| 出站信号机 | starting signal | departure signal | 发车控制。 |
| 调车信号机 | shunting signal | shunt signal | 调车作业。 |
| 道岔转辙设备 | switch machine | point machine | 北美/国际差异。 |
| 转辙机 | point machine | switch machine | 道岔转换设备。 |
| 轨道电路 | track circuit | -- | 列车占用检测。 |
| 计轴 | axle counter | axle counting system | 列车占用检测。 |
| 应答器 | balise | transponder | ETCS/CTCS 常用 balise。 |
| 地面设备 | wayside equipment | trackside equipment | 信号/通信轨旁。 |
| 车载设备 | on-board equipment | onboard equipment | 车上系统。 |
| 列车检测 | train detection | train occupancy detection | 占用检测。 |
| 联锁 | interlocking | -- | 进路、道岔、信号安全逻辑。 |
| 计算机联锁 | computer-based interlocking | CBI | 系统缩写。 |
| 继电联锁 | relay interlocking | -- | 继电器系统。 |
| 进路 | route | train route | 联锁语境。 |
| 进路锁闭 | route locking | -- | 联锁条件。 |
| 接近锁闭 | approach locking | -- | 联锁条件。 |
| 保护区段 | overlap | protection section | 信号安全余量。 |
| 侧面防护 | flank protection | -- | 联锁安全。 |
| 闭塞 | block signalling | block system | 行车间隔控制。 |
| 自动闭塞 | automatic block signalling | ABS | 铁路闭塞。 |
| 半自动闭塞 | semi-automatic block signalling | -- | 闭塞方式。 |
| 固定闭塞 | fixed block | fixed-block signalling | CBTC/传统对比。 |
| 移动闭塞 | moving block | moving-block signalling | CBTC 常见。 |
| 准移动闭塞 | quasi-moving block | -- | 部分系统语境。 |
| 闭塞分区 | block section | block | 区段划分。 |
| 列车间隔 | train separation | headway | 安全间隔用 train separation，运营间隔用 headway。 |
| 追踪间隔 | headway | minimum headway | 运营能力。 |
| 行车许可 | movement authority | MA | ETCS/CBTC/CTCS。 |
| 速度码 | speed code | coded speed | 轨道电路/列控。 |
| 列车控制 | train control | -- | 总称。 |
| 列车自动控制 | automatic train control | ATC | 系统总称。 |
| 列车自动防护 | automatic train protection | ATP | 安全防护。 |
| 列车自动运行 | automatic train operation | ATO | 自动驾驶。 |
| 列车自动监控 | automatic train supervision | ATS | 调度监控。 |
| 基于通信的列车控制 | Communications-Based Train Control | CBTC | 城市轨道。 |
| 中国列车运行控制系统 | Chinese Train Control System | CTCS | 中国铁路。 |
| 欧洲列车控制系统 | European Train Control System | ETCS | ERTMS 子系统。 |
| 欧洲铁路交通管理系统 | European Rail Traffic Management System | ERTMS | 欧洲互操作。 |
| 无线闭塞中心 | Radio Block Centre | RBC | ETCS。 |
| 调度集中 | centralized traffic control | CTC | 铁路调度。 |
| 行车调度 | train dispatching | traffic control | 调度专业。 |
| 调度员 | train dispatcher | traffic controller | 地区和系统有差异。 |
| 行车指挥 | train operation control | traffic control | 运营组织。 |
| 车地通信 | train-to-wayside communication | train-ground communication | 城轨/铁路通信。 |
| 无线通信 | radio communication | wireless communication | 通信系统。 |
| 铁路移动通信系统 | GSM-R | Global System for Mobile Communications - Railway | 标准缩写保留。 |
| LTE-M | LTE-M | metro LTE; LTE for metro | 城轨专用 LTE 语境需说明。 |

| 数据通信网 | data communication network | DCN | 通信网络。 |
| 专用电话 | dedicated telephone | private telephone | 通信系统。 |
| 调度电话 | dispatching telephone | dispatcher telephone | 调度通信。 |
| 广播系统 | public address system | PA system | 车站/列车广播。 |
| 乘客信息系统 | passenger information system | PIS | 客运服务系统。 |
| 闭路电视 | closed-circuit television | CCTV | 视频监控。 |
| 综合监控系统 | integrated supervisory control system | ISCS | 城轨综合监控。 |
| 环境与设备监控系统 | building automation system | BAS | 城轨常用 BAS。 |
| 火灾自动报警系统 | fire alarm system | FAS | 城轨常用 FAS。 |
| 电力监控系统 | power supervisory control system | PSCADA | 供电监控。 |
| 屏蔽门控制系统 | platform screen door control system | PSD control system | 站台门。 |
| 自动售检票系统 | automatic fare collection system | AFC | 票务系统。 |
