# Model Railway Glossary

本表用于模型铁路、火车模型、DCC 数码控制、数字指挥站和模型铁路布局文档。若与真实铁路工程术语冲突，本表优先适用于模型铁路语境。

| English | 推荐中文 | 可选译法 | 说明 |
| --- | --- | --- | --- |
| model railway | 模型铁路 | 火车模型 | 技术文档中优先用“模型铁路”。 |
| model railroad | 模型铁路 | 火车模型 | 北美用语，按模型铁路处理。 |
| model railway layout | 模型铁路布局 | 布局 | 指玩家搭建的轨道、供电、场景和控制系统整体。 |
| command station | 数字指挥站 | 控制站 | DCC/多协议系统核心控制设备。 |
| cab | 操控台 | 控制手柄 | ECoS 双操控台语境用“操控台”。 |
| throttle | 控制手柄 | 调速手柄 | 指速度控制旋钮或手柄，不译作“油门”。 |
| decoder | 解码器 | -- | 包括机车解码器、配件解码器等。 |
| locomotive decoder | 机车解码器 | -- | 移动解码器语境。 |
| accessory decoder | 配件解码器 | -- | 道岔、信号机等固定设备控制。 |
| Hyperlight | Hyperlight 灯光效果 | -- | SoundTraxx 灯光效果产品名，保留 Hyperlight。 |
| lighting effect | 灯光效果 | -- | DCC 解码器功能输出的灯光模拟效果。 |
| function output | 功能输出 | -- | 解码器上由功能键控制的灯光/附件输出。 |
| Rule 17 dimmable headlight | 规则 17 调光前照灯 | Rule 17 调光模式 | 北美机车前照灯调光语境；保留 Rule 17。 |
| grade crossing logic | 平交道口逻辑 | 道口逻辑 | SoundTraxx 灯光效果中由鸣笛/气喇叭触发的道口闪灯逻辑，不译作“坡度交叉逻辑”。 |
| crossing logic | 道口逻辑 | 平交道口逻辑 | 表格列名可用短译“道口逻辑”。 |
| ditch light | 道口闪灯 | 防撞辅助灯 | 北美机车下部辅助警示灯；模型铁路灯光效果中常与平交道口逻辑联动，不译作“沟渠灯/沟灯”。 |
| Type I Ditch Light | I 型道口闪灯 | -- | SoundTraxx 灯光效果名；不闪烁时恢复常亮。 |
| Type II Ditch Light | II 型道口闪灯 | -- | SoundTraxx 灯光效果名；不闪烁时关闭。 |
| Mars Light | Mars Light（马尔斯灯） | 马尔斯摆动警示灯 | 历史机车摆动警示灯/品牌化名称，不译作“火星之光”。 |
| Pyle Gyralite | Pyle Gyralite（摆动警示灯） | Gyralite 摆动灯 | 品牌化机车摆动警示灯名称，首次可保留英文。 |
| oscillating headlight | 摆动前照灯 | 摇摆前照灯 | 灯束摆动扫描的前照灯效果。 |
| rotary beacon | 旋转警示灯 | 旋转信标灯 | 机车顶部旋转灯光警示效果；模型铁路语境优先用“警示灯”。 |
| Western-Cullen D312 Rotary Beacon | Western-Cullen D312 旋转警示灯 | D312 旋转警示灯 | SoundTraxx 灯光效果名，保留型号。 |
| Prime Stratolite | Prime Stratolite（分步旋转警示灯） | Stratolite 旋转警示灯 | 由多灯顺序闪烁形成分步旋转感的警示灯效果。 |
| strobe | 频闪灯 | -- | 氙气频闪/警示灯效果。 |
| single flash strobe | 单闪频闪灯 | -- | SoundTraxx 灯光效果名。 |
| double flash strobe | 双闪频闪灯 | -- | SoundTraxx 灯光效果名。 |
| FRED | FRED（列尾闪光装置） | flashing rear-end device | 北美列车尾端闪光装置，首次出现可括注英文全称。 |
| engine exhaust flicker | 发动机排气闪烁 | 排气火光闪烁 | 灯光模拟效果，不是发动机声音。 |
| firebox flicker | 火箱火光闪烁 | 火箱闪烁效果 | 蒸汽机车火箱灯光模拟效果，不宜只译成“火箱闪烁”。 |
| smart firebox flicker | 智能火箱火光闪烁 | -- | 与司炉铲煤声同步的火箱灯光效果。 |
| Dyno-Light | Dyno-Light（发电机灯光） | 发电机灯光效果 | SoundTraxx 灯光效果名，保留 Dyno-Light。 |
| mixer level | 混音级别 | 混音电平 | 声音解码器音频混音器参数，不译作“混合器级别”。 |
| primary audio mixer | 主要音频混音器 | 主混音器 | SoundTraxx 声音 CV 语境。 |
| secondary mixer | 辅助混音器 | 次级混音器 | SoundTraxx 声音 CV 语境。 |
| master volume | 主音量 | 总音量 | 声音解码器总音量控制。 |
| airhorn | 气喇叭 | -- | 柴油/现代机车喇叭声效。 |
| whistle | 汽笛 | -- | 蒸汽机车汽笛声效；不要泛译为“口哨”。 |
| bell | 铃声 | 警铃 | 声效/功能键语境优先“铃声”。 |
| coupler clank | 车钩撞击声 | 车钩碰撞声 | 声音解码器声效。 |
| brake squeal | 制动尖鸣 | 制动尖叫声 | 声效名，优先“制动尖鸣”。 |
| brake release | 制动释放 | -- | 声音解码器制动释放声效。 |
| rod clank | 连杆敲击声 | 侧杆敲击声 | 蒸汽机车连杆/侧杆机械声。 |
| side rod clank | 侧杆敲击声 | 连杆敲击声 | 需区分 side rod 时优先“侧杆”。 |
| snifter valve | 吸气阀 | -- | 蒸汽机车部件/声效，不译作“嗅探阀/吸枪阀”。 |
| Johnson bar | 换向杆 | -- | 蒸汽机车换向机构操纵杆，不译作“约翰逊酒吧”。 |
| power reverse | 动力逆转器 | -- | 蒸汽机车动力换向装置，不译作“动力反向”。 |
| Johnson Bar/Power Reverse | 换向杆/动力逆转器 | -- | SoundTraxx 声效/CV 名。 |
| pop valve | 安全阀 | -- | 蒸汽机车安全阀声效，不译作“弹出阀”。 |
| pop valve blow off | 安全阀吹放 | -- | 安全阀起跳排汽声效。 |
| blower draft | 鼓风引风 | 鼓风通风 | 蒸汽机车火箱/烟箱鼓风形成引风的声效。 |
| water stop | 止水阀 | 加水停靠 | SoundTraxx 功能键声效中优先译作“止水阀”；叙述列车停靠加水过程时可解释为加水停靠。 |
| injector | 注水器 | -- | 蒸汽机车锅炉注水器，不译作“喷油器”。 |
| tender | 煤水车 | -- | 蒸汽机车随挂车，不译作“招标/补给船”。 |
| glad hand | 制动软管接头 | 空气制动软管接头 | 北美铁路空气制动软管端部连接件，不按字面译。 |
| oil can | 油壶 | -- | 司炉维护声效，不译作“油罐”。 |
| grease gun | 黄油枪 | 注脂枪 | 维护声效。 |
| Fireman Fred | 司炉 Fred | -- | SoundTraxx 蒸汽机车拟人化司炉声效角色，保留 Fred。 |
| booster | Booster | 功率放大器 | ESU/Märklin 文档保留 Booster。 |
| turnout | 道岔 | -- | 模型铁路和真实铁路均可用。 |
| accessory | 配件 | 附件 | 模型铁路控制对象，如道岔、信号机等。 |
| route | 进路 | 路由 | 控制若干配件按预设状态联动时用“进路”。 |
| track diagram | 轨道图 | -- | 控制面板上的布局图。 |
| two-conductor track | 二轨轨道 | 二轨系统 | 不译作“两导体轨道”。 |
| two-rail track | 二轨轨道 | 二轨系统 | 与 two-conductor track 同义。 |
| three-conductor track | 三轨轨道 | 三轨系统 | Märklin 三轨/中间导体系统语境。 |
| three-rail track | 三轨轨道 | 三轨系统 | 不译作“三导体轨道”。 |
| centre conductor | 中间导体 | 中央导体 | Märklin 三轨系统的中间触点/导体。 |
| centre rail | 中间导轨 | 中央导轨 | 若原文明确是 rail，可译为中间导轨。 |
| outer rails | 外轨 | 两根外轨 | 三轨系统中的两侧钢轨。 |
| Main track | 主轨道 | 主轨 | 端子、输出和编程语境优先用“主轨道”。 |
| Main-Track | 主轨道 | 主轨 | ECoS 接线端子名。 |
| programming track | 编程轨道 | 编程轨 | 独立低功率编程输出轨道。 |
| Prog-Track | 编程轨道 | 编程轨 | ECoS 接线端子名。 |
| ECoSLink Connect | ECoSLink 接口 | ECoSLink Connector | ECoS 产品接口名，图注和端子标识中保留 ECoSLink，不缩写为 L1/L2/L3。 |
| ECoSLink Connector | ECoSLink 接口 | ECoSLink Connect | ECoS 产品接口名，按接口定义处理。 |
| ECoSLink Extend | ECoSLink 扩展 | -- | ECoS 产品接口名，图注中保留 ECoSLink。 |
| Programming on the Main | 主轨编程 (POM) | POM | DCC POM，不译作“主控上编程”。 |
| Programming On the Main | 主轨编程 (POM) | POM | 同上。 |
| N gauge | N 比例 | -- | 用户约定：模型比例译作“比例”，不用“N 规”。 |
| H0 gauge | H0 比例 | -- | 用户约定：不用“H0 规”。 |
| G gauge | G 比例 | -- | 用户约定：不用“G 规”。 |
| 1 gauge | 1 号比例 | -- | 用户约定：不用“1 号规”。 |
| Gauge 1 | 1 号比例 | -- | 句首或产品名语境。 |
| loco | 机车 | -- | 英文缩写，中文正文不保留 loco。 |
| loco image | 机车图像 | 机车图标 | ECoS Web 界面中多为机车图标/图像。 |
